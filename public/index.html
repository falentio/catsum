Saestu

saestu is picsum.photo clone, but it has less complexity, saestu writen in Typescript, and runed using Deno not Nodejs. Saestu not using database to store image information, only using single csv file.
---
Swipe from right to left if you wanna see some sample image.
Endpoints:
  • square photo: 
    https://{{hostname}}/200
  • fixed length photo:
    width: 300
    height: 200 
    https://{{hostname}}/300/200
  • grayscaled photo: 
    https://{{hostname}}/300/200?grayscale
  • blured photo:
    blur range is from 1 to 20 
    https://{{hostname}}/300/200?blur=2
  • fixed format photo:
    default is jpg, can be either avif, png, jpg, or webp 
    https://{{hostname}}/300/200.png
  • fixed format photo:
    default is jpg, can be either avif, png, jpg, or webp 
    https://{{hostname}}/300/200.png
  • using seed: 
    https://{{hostname}}/seed/gatitios/300/200.png
  • using id: 
    https://{{hostname}}/id/1/300/200.png
  • using dynamic seed:
    acceptable params is neither of user-agent, ip, date, day, month, and referer
    https://{{hostname}}/seed-with/referer,user-agent/300/200.png
  • list:
    get all
    https://{{hostname}}/list
    get 100 from page 1 
    https://{{hostname}}/list?limit=100
    get 100 from page 2
    https://{{hostname}}/list?limit=100&page=2

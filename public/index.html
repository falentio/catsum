Catsum

By: Kevin Falentio.
Catsum is picsum.photo clone, but it has less complexity, Catsum writen in Typescript, and Deno. Catsum not using database to store image information, only using single csv file.

Endpoints:
  • square photo: 
    {{resolveUrl("/200")}}
  • fixed length photo:
    width: 300
    height: 200 
    {{resolveUrl("/300/200")}}
  • grayscaled photo: 
    {{resolveUrl("/300/200?grayscale")}}
  • blured photo:
    blur range is from 1 to 20 
    {{resolveUrl("/300/200?blur=2")}}
  • fixed format photo:
    default is jpg, can be either of avif, png, jpg, or webp 
    {{resolveUrl("/300/200.png")}}
  • using seed: 
    {{resolveUrl("/seed/gatitios/300/200.png")}}
  • using id: 
    {{resolveUrl("/id/1/300/200.png")}}
  • using dynamic seed:
    acceptable params is either of user-agent, ip, date, day, month, and referer
    {{resolveUrl("/seed-with/referer,user-agent/300/200.png")}}
  • list:
    get all
    {{resolveUrl("/list")}}
    get 100 from page 1 
    {{resolveUrl("/list?limit=100")}}
    get 100 from page 2
    {{resolveUrl("/list?limit=100&page=2")}}
